程序功能：电脑通过手机上网，适合笔记本用户（不用买无线上网卡了）


******* 软硬件需求 *******
手机/终端：支持Java，支持蓝牙，支持web(not wep)上网（gpre/edge/...）。
电脑：蓝牙适配器（内置/外接），
		  Windows XP sp1以上（支持蓝牙协议），
          Java(jre)。
************************


安装方法：

手机：将angent.jar（手机程序）发送到手机，并运行，确认对蓝牙和网络的
访问权限。

电脑：检查蓝牙服务是否启动：我的电脑-管理-服务-Bluetooth Support Service
		  请把电脑与手机配对（配对方法请百度）
		  确认电脑已经安装Java，没安装的话。。（http://java.sun.com）
		  运行webagentpc.jar（电脑程序）
		  启动浏览器，设置代理上网：IE的代理设置 
				工具-Internet选项-连接-局域网设置-为Lan使用代理服务器-地址：127.0.0.1-端口8080-确定
		  在地址栏输入网址，并耐心等待——
		  
		  
******* 注意 *******
并非所有网页都可打开（比如WebQQ就不好用），视手机型号而定。
